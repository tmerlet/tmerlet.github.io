PROJECT 2

Create a photo sharing application.

Small team project, lots of toys like Webcams and facebook.

Users can register and create albums and photos. The app allows multiple file uploads


I used Rails, Jquerui and Ajax, incorporated the use of a webcam to capture profile pics when signing up, and the ability to share images on facebook. I created a thumbs up / thumbs down button with Ajax, so that users can vote on pictures.
